# Bento Grid Background Images Guide

This guide will help you download the right images for each bento card.

## Required Images

You need to download 5 images and save them in this folder with these exact names:
- `live-markets.jpg`
- `find-bars.jpg`
- `watch-parties.jpg`
- `bet-watch.jpg`
- `live-events.jpg`

## Image Requirements
- **Resolution:** 1920x1080 or higher
- **Format:** JPG or PNG
- **Orientation:** Landscape
- **Theme:** Dark/moody photos work best with overlays

## Quick Download Instructions

### From Unsplash:
1. Search for the suggested terms
2. Click on an image you like
3. Click the "Download" button (free, no account needed)
4. Rename the file to match the required name
5. Move it to this folder

### From Pexels:
1. Search for the suggested terms
2. Click on an image you like
3. Click "Free Download" → Select "Original" size
4. Rename the file to match the required name
5. Move it to this folder

---

## Tips for Choosing Images

✅ **DO:**
- Choose dark or moody images (work better with overlays)
- Pick high-resolution images (1920x1080+)
- Select images that clearly represent the concept
- Prefer professional-looking photos

❌ **DON'T:**
- Use overly bright images (text won't be readable)
- Choose images with important details in the center (will be covered by text)
- Pick low-resolution images (will look pixelated)
- Use images with watermarks

---

## Testing Your Images

After downloading and placing the images:

1. Start the dev server: `npm run dev`
2. Navigate to `http://localhost:3000`
3. Scroll to "How it works" section
4. Check that:
   - All images load properly
   - Text is readable over each image
   - Images represent the concepts well
   - No layout issues on mobile/tablet/desktop
